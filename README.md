# 食用方法 （github_actions）

 1.点击右上角 **fork** 按钮 
 
 2.在自己仓库中打开此项目
 
 3.配置环境变量
 
 4.点击右上角 **star** 激活actions
 
 5.你可以点击 Actions 标签查看运行的详细状况
